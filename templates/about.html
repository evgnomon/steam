About

This is a demo application showcasing the Hotwire stack with FastAPI:

  • Turbo Drive - Accelerates navigation by intercepting link clicks and form submissions, fetching pages via AJAX, and swapping the body content without a full page reload.
  • Turbo Streams - Delivers page updates over WebSocket, allowing real-time updates to the DOM.
  • Stimulus - A modest JavaScript framework for the HTML you already have, providing controllers that connect to DOM elements.

How Turbo Drive Works

When you click links between pages (like Home and About), Turbo Drive:

  1. Intercepts the click event
  2. Fetches the new page via fetch API
  3. Replaces the <body> content
  4. Updates the browser history

This results in faster navigation without writing any JavaScript!
